## How to Compile

1. Open a terminal or command prompt.
2. Navigate to the directory containing the source file 
3. Compile the code using:

   g++ -std=c++11 -o file name.cpp

  

## How to Run

- On Linux or macOS:
  ./file name

- On Windows:
  filename.exe
